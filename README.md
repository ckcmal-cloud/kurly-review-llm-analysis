# 마켓컬리 신선식품 VOC 분석

> 마켓컬리 신선식품 리뷰 6,441건을 맛·품질·신선도·배송 4개 평가축과 감성으로 분류하고, 카테고리별 불만 원인과 운영 우선순위를 도출한 리뷰 기반 VOC 분석

## 프로젝트 목표

> 신선식품은 재구매 빈도가 높은 상품군으로, 온라인 구매 환경에서는 리뷰가 구매 및 재구매 결정에 중요한 역할을 하므로
카테고리별 핵심 만족 및 불만 요인을 구조화하여 리뷰 기반 운영 우선순위 도출, 고객 경험 및 리텐션 개선 가능성을 검토
(리뷰 → 구매결정 → 재구매 → 리텐션)

## Dataset

- 원본 리뷰 데이터: 6,441건
- 고유 상품 수: 121개
- 카테고리 수: 6개
- 수집 데이터 기간: 2021-12 ~ 2026-04
- 평가축: 맛 · 품질 · 신선도 · 배송
- 감성 분류: 긍정 · 중립 · 부정

## Analysis

### 1. 리뷰 데이터 구축 및 전처리

- API 기반 리뷰 데이터 수집
- 상품번호 기반 카테고리 매핑
- 토마토 subtype 세분화
- 리뷰 텍스트 전처리 및 파생변수 생성

### 2. VOC 구조 분석

- 맛·품질·신선도·배송 4축 분류
- 감성 분류
- 카테고리별 평가축 분석
- 부정 리뷰 원인 분석
- 긍정 리뷰 셀링포인트 분석

### 3. 검증 및 운영 우선순위 도출

- LLM 검증 비교
* 카이제곱 검정 및 효과크기 분석
* 운영 우선순위 지수 산출
* P1·P2·P3 운영 우선순위 도출

## 주요 결과

- 전체 리뷰의 67.6%가 긍정, 4.4%가 부정으로 나타남
- 맛 언급률이 50.0%로 가장 높았으며 품질(20.4%), 신선도(19.0%), 배송(13.7%) 순으로 나타남
- 생연어는 신선도(33.3%), 계란은 신선도(35.5%)와 배송(29.8%) 언급 비중이 높게 나타남
- 부정 리뷰 원인은 카테고리별로 차이를 보였으며, 계란은 배송(61%), 생연어는 신선도(58%), 소고기등심은 맛(43%) 이슈가 중심으로 나타남
- 카테고리별 평가축 차이는 모두 통계적으로 유의했으며, 신선도 축이 가장 큰 차이를 보임(Cramér's V=0.270)
- 부정률, 리뷰 비중, 중요도를 반영한 운영 우선순위 분석 결과 소고기등심(P1), 딸기·토마토(P2)가 우선 대응 대상으로 도출됨

## 인사이트

- 온라인 신선식품 구매에서는 리뷰가 구매 및 재구매 결정에 중요한 역할을 함
- 전체 만족도보다 카테고리별 핵심 평가축과 불만 원인을 구분하는 것이 중요
- 부정률뿐 아니라 리뷰 규모와 중요도를 함께 고려한 우선순위 관리가 필요
- 리뷰 기반 VOC 분석을 통해 고객 이탈 방지 및 리텐션 개선 전략 수립 가능
- 비정형 리뷰를 평가축 단위로 구조화하면 운영 의사결정에 활용할 수 있음
